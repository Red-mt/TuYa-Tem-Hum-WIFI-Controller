#项⽬简介
1、基于Arduino设计的工业除湿控制器，WIFI联网，报警上报，适用于大型除湿机（如滑冰场除湿）； 
2、手机可远程设置开启阈值；
3、市电供电（220V）,可根据需要更换对应的AC-DC模块；
4、可实时查看当前温湿度； 
5、温湿有变化时自动上报（温度0.1℃变化、湿度1%变化）；
